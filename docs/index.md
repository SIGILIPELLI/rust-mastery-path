# Rust Mastery Path

A structured, module-wise Rust training program that takes you from your first
`cargo new` to production-grade, master-level systems engineering — with
runnable code snippets in every module and a hands-on project at the end of
each level.

## How the program is organized

| Level | Focus | Modules |
|-------|-------|---------|
| [Level 1 · Entry](level-1/index.md) | Language foundations & ownership basics | 9 topics + 1 project |
| [Level 2 · Intermediate](level-2/index.md) | Traits, lifetimes, idiomatic Rust | 9 topics + 1 project |
| [Level 3 · Advanced](level-3/index.md) | Concurrency, async, web services, databases | 9 topics + 1 project |
| [Level 4 · Master](level-4/index.md) | Production systems & architecture | 9 topics + 1 capstone |

## How to use this site

- Work through each level in order — later modules assume earlier ones.
- Every topic page has runnable code snippets — copy them into a local Cargo
  project (`cargo new my_project`) and run with `cargo run`, or paste into the
  Rust Playground.
- Each level ends with a project that combines everything learned in that level.
- Use the search bar (top of the page) to jump straight to a topic.

Start here → [Level 1 · Entry](level-1/index.md)
